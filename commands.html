Name Description Example
echo echo the STRING OR STRINGS to the standard output sp2b@rpi02:~$ echo baseball "baseball"
ls The ls function will list the contents of directory. Information about the files in the directory sp2b@rpi02:~$ ls "Website Commands Readme"
ps The ps function lists the current running processes and their PIDs sp2b@rpi02:~$ ps -s "618"
kill This function terminates the running processes sp2b@rpi02:~$ kill -51823
hostname This obtains the Domain Name System for the computer sp2b@rpi02:~$ hostname -myComputers
sleep Creates a dummy job that is able to delay the exacution of certain programs sp2b@rpi02:~$ sleep 250
cd This command copies the source File to the Destination file or the Destoyer sp2b@rpi02:~$ cd "directory 02"
cp This command coppies the Source File to the Destination file of the Directory sp2b@rpi02:~$ cp "readme readme2"
date This command displays the current date and time of the Linux operating system sp2b@rpi02:~$ date "Tues May 28 22:00 PDT 2019"
uname -a This command reports information about a computer's software and hardwar$ sp2b@rpi02:~$ uname -o "system of operation"
